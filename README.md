# Business-Performance-Analytics-Dashboard
## Retail Analytics Dashboard | Excel & Power Query
### Project Overview

This project is an end-to-end Retail Analytics Dashboard built in Microsoft Excel using a D-Mart-inspired sales dataset. The objective was to transform raw transactional data into actionable business insights through data cleaning, financial analysis, visualization, and dashboarding.

### Key Highlights
* Analyzed 100+ retail transactions covering 12 months of sales activity.
* Performed comprehensive data cleaning and transformation using Power Query, including handling duplicates, missing values, inconsistent formatting, and outliers.
* Built Pivot Tables, Pivot Charts, Lookup Functions, and Calculated Fields for detailed business analysis.
* Developed an interactive dashboard featuring KPI Cards, Slicers, Sales Trends, Regional Analysis, Customer Segmentation, and Payment Mode Insights.
* Calculated key financial metrics such as Net Revenue, Profit, Profit Margin, Average Purchase Value, and Transaction Counts.
* Generated business recommendations through trend analysis and performance evaluation.
